SEJARAH


Pada 2009 UNEJ membuka sebuah program studi setara fakultas, yakni Program Studi Sistem Informasi (PSSI).
Memiliki 2 program studi: Sistem Informasi danTeknologi Informasi, maka dipandang perlu untuk merestrukturisasi PSSI (setara fakultas) menjadi sebuah Fakultas Ilmu Komputer. Pada tahun 2017 fasilkom membuka program studi baru yakni informatika.
